## Preguntas y Respuestas de la Entrevista

### Proyecto Events Manager

**P1:** ¿Por que decidiste usar Next.js como framework principal para el proyecto Events Manager en lugar de otras opciones como Vite + React o un framework diferente?

**R1:** Por el sistema de rutas y su integracion con Amplify

**P2:** Veo que implementaste funcionalidades de firma digital (react-signature-canvas) y generacion de PDFs (jspdf) en el Events Manager. ¿Cual fue el razonamiento detras de integrar estas caracteristicas especificamente en lugar de usar servicios externos para el manejo de documentos?

**R2:** Sobre todo por el tema de costos, ademas el cliente solo requeria una solucion a medida

### Proyecto Club de Ganadores

**P3:** En el proyecto Club de Ganadores, implementaste animaciones con Framer Motion y funcionalidades de ranking. ¿Que te llevo a elegir Framer Motion sobre otras librerias de animacion como React Spring o CSS puro para las animaciones del ranking?

**R3:** La verdad no tenia mucha experiencia sobre animacion, fue una recomendacion que obtuve

**P4:** ¿Cual fue la razon especifica para permitir la carga de archivos Excel en lugar de usar formularios tradicionales para la entrada de datos en el sistema de ranking?

**R4:** Porque los usuarios finales ya tenian un template en excel donde capturaban su informacion y son muchos datos
